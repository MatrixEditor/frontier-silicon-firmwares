# Known radios

A curated list of known Frontier Smart radio devices. The name can be split up into different parts:

```bnf
<MODULE_TYPE> "-" <MODULE_VERSION> "-" <PRODUCT_CODE>
```

1. Module type: There are 18 module types known so far. Most of them are taken from [here](https://www.electronicsdatasheets.com/manufacturers/frontier-silicon/parts/):

  * `FS1230`:  Chorus 3
  * `FS1235`:  Kino
  * `FS2025`:  Venice 5
  * `FS2026`:  Venice 6
  * `FS2027`:  Venice 7
  * `FS2028`:  Venice 8
  * `FS2029`:  Venice 9
  * `FS2052`:  Verona
  * `FS2230`:  Tuscany
  * `FS2240`:  Roma
  * `FS2340`:  Venice X
  * `FS2445`:  Verona 2
  * `FS4052`:  Venus
  * `FS4053`:  Venus 2I-L
  * `FS4230`:  Neptune
  * `FS4240`:  Ceres
  * `FS4255`:  Venus-H2
  * `FS5332`:  Minuet

2. Module version: Some modules contain different version numbers. For instance, the Venice 6 module has `0500` as well as `0200` definitions, which can be translated into `6.5` or `6.2`.

3. Product code: The last parts specifies the product code that is used to identify the product.
